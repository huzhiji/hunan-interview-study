## 学习系统使用方式

### 每日学习 (自动生成)
- 命令：`node daily_system.js today`
- 输出：2句金句 + 1个热点 + 1位人物 + 1个措施
- 时间：每天8:00自动推送

### 每周测验 (自动生成)
- 命令：`node daily_system.js quiz [数量]`
- 输出：知识测验题 + 模拟面试题
- 时间：每周日晚7:00自动推送

### 常用命令
```bash
# 查看今日任务
node daily_system.js today

# 标记今日完成
node daily_system.js complete

# 生成测验
node daily_system.js quiz 5

# 生成模拟题
node daily_system.js interview

# 查看进度
node daily_system.js progress
```
